Job Application Tracker

The Job Application Tracker is a simple and user-friendly web application that helps users manage their job search process efficiently. It allows users to track the total number of job applications, interviews, and rejected applications in one place.

This project is built to solve a real-life problem—keeping job application data organized without using spreadsheets or complex tools. The clean interface makes it easy for users to quickly understand their progress.

Key Features:

Track total job applications
Monitor interview status
Keep record of rejected applications
Clean and responsive design

Technologies Used:
HTML,CSS ,Tailwind CSS, JavaScript

Through this project, I improved my skills in responsive design, layout structuring, and dynamic data handling using JavaScript. It also reflects my ability to build practical and real-world web applications.
